# Глава 26. Лоренцевы преобразования полей: Учебный конспект

Данный конспект составлен на основе материалов 26-й главы «Феймановских лекций по физике» и охватывает релятивистское описание электромагнитных полей, их трансформационные свойства и уравнения движения заряженных частиц.

---

## 1. Ключевые понятия и теоретические основы

### Четырехмерный потенциал ($A_\mu$)
Потенциал в электродинамике представляет собой четырехвектор $A_\mu = (\phi, \mathbf{A})$, где:
*   **$\phi$** — временная компонента (скалярный потенциал).
*   **$\mathbf{A}$** — три пространственные компоненты (векторный потенциал).

Для точечного заряда, движущегося равномерно со скоростью $v$ вдоль оси $x$, потенциалы в точке $(x, y, z)$ определяются через координаты относительно текущего положения заряда $P(vt, 0, 0)$.

### Запаздывающее и проекционное положения
*   **Запаздывающее положение ($P'$)**: Точка, в которой находился заряд в момент времени $t' = t - r'/c$. Именно состояние заряда в этой точке определяет поле в точке наблюдения в настоящий момент $t$.
*   **Проекционное положение ($P_{\text{proj}}$)**: Воображаемая точка, в которой оказался бы заряд в момент $t$, если бы он продолжал движение с постоянной скоростью $v'$, зафиксированной в запаздывающий момент. Потенциалы произвольно движущегося заряда в точности равны потенциалам, которые создал бы гипотетический заряд, находящийся в $P_{\text{proj}}$ и движущийся равномерно.

### Электрическое и магнитное поля движущегося заряда
Для заряда, движущегося с постоянной скоростью:
1.  **Направление**: Электрическое поле $\mathbf{E}$ направлено радиально от **текущего** (а не запаздывающего) положения заряда.
2.  **Конфигурация (Лоренцево сокращение)**: Силовые линии «сжимаются» вдоль направления движения. Поле по бокам (перпендикулярно движению) усиливается в $1/\sqrt{1-v^2/c^2}$ раз, а впереди и позади — ослабевает.
3.  **Магнитное поле**: Определяется формулой $\mathbf{B} = \frac{\mathbf{v} \times \mathbf{E}}{c^2}$. Его силовые линии образуют окружности вокруг линии движения.

### Электромагнитный тензор ($F_{\mu\nu}$)
Электрическое и магнитное поля не являются отдельными четырехвекторами. Они представляют собой компоненты единого **антисимметричного тензора второго ранга** в четырехмерном пространстве. Этот объект имеет 6 независимых компонент.

| Компонента тензора | Соответствующее поле |
| :--- | :--- |
| $F_{xt}, F_{yt}, F_{zt}$ | Компоненты электрического поля ($E_x, E_y, E_z$) |
| $F_{xy}, F_{yz}, F_{zx}$ | Компоненты магнитного поля ($-B_z, -B_x, -B_y$) |

## 2. Преобразования Лоренца для полей

Если известны поля в неподвижной системе $S$, их значения в системе $S'$, движущейся со скоростью $v$ вдоль оси $x$, вычисляются по следующим правилам (при $c=1$):

| Электрическое поле | Магнитное поле |
| :--- | :--- |
| $E_x' = E_x$ | $B_x' = B_x$ |
| $E_y' = \frac{E_y - vB_z}{\sqrt{1-v^2}}$ | $B_y' = \frac{B_y + vE_z}{\sqrt{1-v^2}}$ |
| $E_z' = \frac{E_z + vB_y}{\sqrt{1-v^2}}$ | $B_z' = \frac{B_z - vE_y}{\sqrt{1-v^2}}$ |

### Обобщенная форма (через параллельные и перпендикулярные компоненты)
*   $E_\parallel' = E_\parallel$, $B_\parallel' = B_\parallel$ (компоненты вдоль направления движения не меняются).
*   $E_\perp' = \frac{(\mathbf{E} + \mathbf{v} \times \mathbf{B})_\perp}{\sqrt{1-v^2/c^2}}$
*   $B_\perp' = \frac{(\mathbf{B} - \frac{\mathbf{v} \times \mathbf{E}}{c^2})_\perp}{\sqrt{1-v^2/c^2}}$

---

## 3. Релятивистское уравнение движения

Сила Лоренца в обычном виде: $\mathbf{F} = q(\mathbf{E} + \mathbf{v} \times \mathbf{B})$.
Релятивистское уравнение движения:
$$\frac{d}{dt} \left( \frac{m_0 \mathbf{v}}{\sqrt{1-v^2/c^2}} \right) = \mathbf{F}$$

В четырехмерной форме это уравнение принимает «элегантный» вид:
$$m_0 \frac{d^2x_\mu}{ds^2} = f_\mu = q u_\nu F_{\mu\nu}$$
где:
*   $s$ — собственное время ($ds = dt \sqrt{1-v^2/c^2}$).
*   $u_\nu$ — четырехмерная скорость.
*   $f_\mu$ — четырехмерная сила.

## 4. Проверочные вопросы

### Краткие ответы
1.  **Как направлено электрическое поле заряда, движущегося с постоянной скоростью?**
    *   *Ответ:* Радиально от его текущего (настоящего) положения.
2.  **Почему нельзя получить всю электродинамику только из закона Кулона и преобразований Лоренца без дополнительных допущений?**
    *   *Ответ:* Необходимо дополнительно постулировать, что потенциалы образуют четырехвектор и что влияние заряда зависит только от его состояния в запаздывающий момент времени.
3.  **Какое свойство тензора $F_{\mu\nu}$ объясняет, почему компоненты типа $F_{xx}$ или $F_{tt}$ равны нулю?**
    *   *Ответ:* Антисимметричность тензора ($F_{\mu\nu} = -F_{\nu\mu}$).
4.  **Что такое «собственное время» частицы?**
    *   *Ответ:* Интервал времени, измеренный по часам, движущимся вместе с частицей; инвариантная величина $ds = dt \sqrt{1-v^2/c^2}$.

### Вопросы для глубокого изучения (Эссе)
1.  **Нарушение третьего закона Ньютона в релятивистской электродинамике.** Опишите ситуацию с двумя протонами, движущимися по пересекающимся перпендикулярным траекториям. Почему силы взаимодействия между ними не равны и не противоположны?
2.  **Физический смысл проекционного положения.** Объясните, почему для вычисления потенциалов произвольно движущегося заряда удобно использовать концепцию $P_{\text{proj}}$.
3.  **Практические ограничения измерения скорости через магнитное поле Земли.** На примере самолета объясните, почему теоретическая возможность измерения скорости по наведенному электрическому полю $\mathbf{v} \times \mathbf{B}$ труднореализуема на практике.

## 5. Типичные ошибки

*   **Ожидание, что поля направлены от запаздывающего положения:** Часто полагают, что раз сигнал идет от $P'$, то и вектор $\mathbf{E}$ направлен оттуда. На самом деле для равномерного движения поле направлено точно от текущего положения $P$.
*   **Стремление представить поля как четырехвекторы:** Попытка сопоставить вектор $\mathbf{E}$ или $\mathbf{B}$ с какой-то четвертой компонентой для создания четырехвектора ошибочна. Поля — это компоненты тензора второго ранга (шестикомпонентного объекта).
*   **Игнорирование множителя $\sqrt{1-v^2/c^2}$ в поперечных компонентах:** При преобразовании полей важно помнить, что меняются не только значения за счет добавления $\mathbf{v} \times \mathbf{B}$, но и масштаб самих поперечных компонент из-за релятивистского фактора.

## 6. Краткий глоссарий

*   **4-вектор потенциала ($A_\mu$):** Единый объект, объединяющий скалярный потенциал $\phi$ и векторный потенциал $\mathbf{A}$.
*   **Запаздывающее время ($t'$):** Момент времени в прошлом, когда сигнал, движущийся со скоростью света, покинул заряд, чтобы достичь точки наблюдения в момент $t$.
*   **Лоренцево сокращение поля:** Релятивистский эффект, при котором плотность силовых линий движущегося заряда увеличивается в плоскости, перпендикулярной вектору скорости.
*   **Собственное время ($s$ или $\tau$):** Время, измеряемое в системе отсчета, связанной с самой движущейся частицей.
*   **Тензор второго ранга:** Математический объект (в данном контексте $F_{\mu\nu}$), компоненты которого преобразуются при переходе между системами отсчета как произведение компонент двух векторов.
*   **Четырехмерная сила (4-сила, $f_\mu$):** Релятивистское обобщение силы, временная компонента которого связана с мощностью (скоростью изменения энергии).
