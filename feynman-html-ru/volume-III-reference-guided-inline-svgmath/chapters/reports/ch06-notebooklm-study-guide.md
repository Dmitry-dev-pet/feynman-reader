# Глава 6. СПИН ОДНА ВТОРАЯ: Учебное пособие

Данное пособие синтезирует основные теоретические положения шестой главы третьего тома «Фейнмановских лекций по физике», посвященной математическому описанию систем со спином 1/2 и законам преобразования их амплитуд при поворотах системы координат.

---

## 1. Ключевые понятия и теоретические основы

### Преобразование амплитуд
Любое квантовое состояние $|\psi\rangle$ может быть описано через совокупность базисных состояний. Амплитуда перехода между состояниями выражается суммой:
$$\langle\chi|\psi\rangle = \sum_i \langle\chi|i\rangle\langle i|\psi\rangle$$
При переходе от одного базиса ($S$) к другому ($T$) амплитуды $C_i = \langle iS|\psi\rangle$ преобразуются в $C_j' = \langle jT|\psi\rangle$ с помощью матрицы преобразования $R_{ji}$:
$$C_j' = \sum_i R_{ji}C_i$$
Эта совокупность коэффициентов называется **матрицей поворота**.

### Произвол фазы и нормировка
1.  **Общая фаза:** Умножение всех амплитуд на один и тот же фазовый множитель $e^{i\delta}$ не меняет физических результатов (вероятностей).
2.  **Стандартная форма:** Матрицы поворота приводятся к стандартному виду так, чтобы их детерминант был равен единице ($\Det R = 1$). Это позволяет однозначно определять фазы.

### Свойства пространства и повороты
Матрица поворота зависит только от взаимного расположения приборов (анализаторов) в пространстве, а не от их абсолютной ориентации. Последовательные повороты описываются перемножением матриц:
$$R_{ki}^{US} = \sum_j R_{kj}^{UT}R_{ji}^{TS}$$

### Особенности спина 1/2
Для частиц со спином 1/2 (например, электронов в приборе Штерна — Герлаха) характерны следующие свойства:
*   **Отсутствие нуль-состояния:** Существуют только состояния «вверх» (+) и «вниз» (-).
*   **Поворот на 360°:** При полном повороте вокруг любой оси амплитуды меняют знак ($C' = -C$). Это не меняет физику (вероятности остаются прежними), но является критически важным для математического описания. Первым углом, воспроизводящим исходное физическое состояние, является $\phi = 360^\circ$.

## 2. Матрицы преобразования (Сводные таблицы)

Используя углы Эйлера ($\alpha, \beta, \gamma$), можно описать любой произвольный поворот системы $(x, y, z)$ в $(x', y', z')$.

### Таблица 2.1. Общая матрица поворота $R_{ji}(\alpha, \beta, \gamma)$
| Состояние | $|+S\rangle$ | $|-S\rangle$ |
| :--- | :--- | :--- |
| **$\langle+T|$** | $\cos\frac{\alpha}{2} e^{i(\beta+\gamma)/2}$ | $i\sin\frac{\alpha}{2} e^{-i(\beta-\gamma)/2}$ |
| **$\langle-T|$** | $i\sin\frac{\alpha}{2} e^{i(\beta-\gamma)/2}$ | $\cos\frac{\alpha}{2} e^{-i(\beta+\gamma)/2}$ |

### Таблица 2.2. Амплитуды для поворотов вокруг основных осей на угол $\phi$
| Тип поворота | Матрица (коэффициенты перехода) |
| :--- | :--- |
| **Вокруг оси $z$** | $C_+' = e^{i\phi/2}C_+$ <br> $C_-' = e^{-i\phi/2}C_-$ |
| **Вокруг оси $x$** | $C_+''' = (\cos\frac{\phi}{2})C_+ + i(\sin\frac{\phi}{2})C_-$ <br> $C_-''' = i(\sin\frac{\phi}{2})C_+ + (\cos\frac{\phi}{2})C_-$ |
| **Вокруг оси $y$** | $C_+' = (\cos\frac{\phi}{2})C_+ + (\sin\frac{\phi}{2})C_-$ <br> $C_-' = -(\sin\frac{\phi}{2})C_+ + (\cos\frac{\phi}{2})C_-$ |

## 3. Проверочные вопросы (Short-answer quiz)

1.  **Сколько пучков образуется в приборе Штерна — Герлаха при пропускании частиц со спином 1/2?**
    *   *Ответ:* Два пучка (состояния «вверх» и «вниз»).
2.  **Как изменяются модули амплитуд $|C_+|$ и $|C_-|$ при повороте системы координат вокруг оси $z$?**
    *   *Ответ:* Модули остаются неизменными, меняются только фазы амплитуд.
3.  **К какому математическому результату приводит поворот системы на 360° для системы со спином 1/2?**
    *   *Ответ:* Обе амплитуды умножаются на $-1$.
4.  **Чему равна вероятность обнаружить частицу в состоянии «вверх по $x$», если она была приготовлена в состоянии «вверх по $z$»?**
    *   *Ответ:* 50% ($|1/\sqrt{2}|^2$).
5.  **Какое свойство детерминанта используется для приведения матрицы поворота к «стандартной форме»?**
    *   *Ответ:* $\Det R = 1$.

## 4. Эссе: Темы для глубокого изучения

1.  **Симметрия пространства и квантовые амплитуды:** Проанализируйте, каким образом предположение об изотропности пространства (отсутствие предпочтительного направления) позволяет вывести конкретные значения коэффициентов матрицы преобразования.
2.  **Абстрактная логика в квантовой механике:** Фейнман называет вывод формул в этой главе «культурным мероприятием». Объясните, почему глубоко абстрактные рассуждения о структуре пространства часто оказываются более точными, чем конкретные физические модели внутреннего устройства частиц.
3.  **Физический смысл фазы:** На примере экспериментов с последовательными приборами Штерна — Герлаха (фиг. 4.5) покажите, почему различие фаз амплитуд (даже при одинаковых модулях) приводит к разным физическим результатам.

## 5. Типичные ошибки при изучении темы

*   **Ошибка $m=1$:** Предположение, что при повороте на 360° амплитуда должна вернуться к исходному значению ($e^{i2\pi} = 1$). Для спина 1/2 это неверно, так как при $m=1$ состояние $+x$ перешло бы в $-x$ при повороте на 180°, что противоречит физике. Правильный коэффициент в экспоненте — $1/2$.
*   **Игнорирование фазового сдвига:** Ошибка считать, что если вероятности одинаковы, то и состояния идентичны. Разница в фазах между $C_+$ и $C_-$ определяет поляризацию частицы в плоскости $xy$.
*   **Путаница с углами Эйлера:** Неверная последовательность поворотов при использовании углов $\alpha, \beta, \gamma$. Согласно источнику, порядок таков: поворот вокруг $z$ на $\beta$, затем вокруг временной оси $x$ на $\alpha$, затем вокруг новой оси $z'$ на $\gamma$.

## 6. Краткий глоссарий

*   **Базисные состояния:** Набор ортогональных состояний, через которые можно выразить любое другое состояние системы.
*   **Матрица преобразования (поворота):** Совокупность коэффициентов, позволяющая пересчитать амплитуды вероятности из одной системы координат в другую.
*   **Спин 1/2:** Характеристика систем (атомов, частиц), которые в неоднородном магнитном поле расщепляются ровно на два пучка.
*   **Углы Эйлера:** Три угла ($\alpha, \beta, \gamma$), описывающие произвольную ориентацию одной системы координат относительно другой через три последовательных поворота.
*   **Фазовый множитель ($e^{i\delta}$):** Комплексное число с модулем, равным единице, на которое может быть умножена амплитуда без изменения вероятности события.
